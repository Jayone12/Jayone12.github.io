JavaScript 시작 (3-3) - 문자열 타입(string type)

2022/04/29 카테고리 JavsScript

텍스트 데이터를 나타내는데 사용하며, 0개 이상의 16비트 유니코드 문자(UTF-16)의 집합으로 전 세계 대부분의 문자 표현이 가능하다.

문자열 표현 방법은 작은 따옴표(‘’), 큰 따옴표(“”), 백틱(``)으로 감싸며 아래와 같다.

❗️자바스크립트의 문자열은 원시타입이며, 변경 불가능한 값이다.


문자열의 불변성에 대해서는 차후에 정리할 예정.


템플릿 리터럴

문자열을 표현하는 방법 중 백틱(``)은 ES6부터 새롭게 생긴 템플릿 리터럴이라는 문자열 표기법이다.
템플릿 리터럴은 문자열, 표현식 삽입, 태그드 템플릿 등 한번에 문자열 처리 기능을 제공하며, 런타임에 일반 문자열로 변환한다.


멀티라인 문자열

일반 문자열 내에서는 줄바꿈이 허용 되지 않는다.

이스케이프 시퀀스(Escape sequence)

이스세이프 시퀀스 의미
\0 Null
\b 벡스페이스
\f 폼 피드: 프린터로 출력할 경우 다음 페이지의 시작 지점으로 이동
\n 줄 바꿈
\t 탭(수평)
\v 탭(수직)
' 작은 따옴표
" 큰 따옴표
\ 백슬래시

표현식 삽입

문자열은 +연산자를 사용해여 연결할 수 있다.


템플릿 리터널의 경우 표현식 삽입은 ${}으로 표현식을 감싸면 된다.
