React 시작하기 (1) - 리액트란 무엇인가?

2022/03/13 카테고리 React

리액트란 유저 인터페이스를 만들기 위한 JavaScript 라이브러리로 현 meta, 전 facebook에서 개발하였다.

리액트의 특징

  • 컴포넌트(component)라는 작고 독립적인 코드 블록을 조합하여 빠르고 효율적으로 화면을 구성한다.
    • 컴포넌트란? - 문서(HTML), 스타일(CSS), 동작(JS)을 합쳐서 내가 만든 일종의 태그
  • 다음에 나타날 화면 일부(노드)를 미리 그려놓고 변경된 화면의 일부(노드)만 수정하는 가상 화면(Virtual DOM) 기술로 화면 출력속도가 빠르다.
  • 단방향 데이터 바인딩으로 데이터 변화에 따른 성능 저하 없이 DOM 객체 갱신이 가능하다. (js → HTML)
  • JSX라는 리액트만의 문법으로 JavaScript와 HTML을 동시에 사용할 수 있어 편리하다.
