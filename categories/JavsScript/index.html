node.js란?

2022/05/16 카테고리 JavsScript

node.js는 Chrome V8 JavaScript 엔진으로 빌드된 JavaScript 런타임이다.

런타임이란?
프로그래밍 언어가 동작하는 환경을 말한다.

브라우저의 JavaScript 엔진에서 동작하는 JavaScript를 브라우저 이외의 환경에서도 동작할 수 있도록하는 실행환경이다.
